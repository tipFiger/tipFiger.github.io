部署Django¶

Django 中简化 Web 开发的快捷函数俯拾皆是，但是如果不能轻松部署站点的话，所有的工具都徒劳无益。Django 从最初就把方便部署当作一个主要目标。

  • 如何使用 WSGI 进行部署
  • 部署清单

如果你是初次接触 Django 或者 Python，我们推荐你先尝试阅读文档:doc:`mod_wsgi`</howto/deployment/wsgi/modwsgi>。在大多数情况，这种部署方式是最简单，最快和最稳定的。
